O que é um repositório ? Exemplos: Dropbox, google drive
- Local: quando os arquivos estam na minha máquina
- Remoto: arquivo na nuvem

git clone - pega do remoto e trás para o local
usar git glone dentro do git clash
git init para cololar a pasta como projeto do github (cria um repositório local)
git status para verificar
git add para adicionar
git commit -m "" (para descrever a alteração).
git log mostra todo o registro de alterações feito no documento
git diff para ver o que foi alterado e confirmar se deseja alterar isso mesmo
git pull origin master (Verifica todos que foram alterados no repositório e trás para o local.
